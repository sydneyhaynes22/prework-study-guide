HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • Elements nestled inside other elements are called child elements
  • The elements that hold all the "children" are known as the parent element
  • HTML builds the frame, CSS defines the style, and JavaScript makes it interactive
  • HTML attributes are always defined in the beginning element never the closing element

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element
  • Three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet
  • Inline CSS: quick and permanent not as flexible as others, requires each inline style individually edited
  • Internal style sheet unlike inline can embed entire stylesheet directly into HTML
  • External CSS style sheet us most common allows CSS rules to be kept in separate file, changes are easier

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git branch: to see local branches
  • git log: show all commits in current branches history
  • Pull Requests (PR): way to ask for changes made to a branch to be merged to another, others can see discuss and review
  • A branch is a copy/version of code that can be worked on independently in isolation

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • To declare a variable use the var keywork, after that give the variable a unique name, than use = to assign data we are storing in it
  • Control flow is the order in which a computer executes code in a script.
  • A for loop or conditional statement automatically execute when the JavaScript file is run
  • A function does not automatically run, you must call a function by name to execute it
  • Boolean: True/False value, these are special keywords and don't need quoote marks
  • Array: structure that allows you to store multiple values in a single reference
  • (===): Checks to see if two values are equal and returns a boolean result true if values are equal and false if they are not equal
